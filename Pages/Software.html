Software

Software is the series of instructions that cause a computer to run. Software directs the actions of the computer's hardware and it's processes. Software is constantly being run on computers and are what the user interacts with and uses computers for. Hardware sets up the stage for the software to be able to run efficiently.

Operating System

The operating system is the program that is run constantly by the computer from startup to shutdown. Operating systems provide some basic functionality to the user and applications that are run on the operating system.

Operating systems are used to communicate between physical devices and software programs and it also allows the computer to connect and communicate with external devices. As well as this, operating systems act as an interface between the user and the hardware.

Utility Software

Utility software is software that allows the user to configure, analyse and optimise software and data in a device. Utility software is often primarily functional and allows the user to change and see files, back up files, run antivirus checks and other functions that are necessary for the upkeep and continual maintenance of a device.

Application Software

Application software is software that performs specific tasks for the user, they provide special functionality to assist the user to complete tasks. Application software allows the user to be productive, anything from excel to a video game. Application software comprises the majority of software that a user may use on a computer.

Some examples of common application software include:

  • Web browsers, such as Chrome or Brave
  • Applications such as Excel or Microsoft Word
  • Music players, such as Spotify
